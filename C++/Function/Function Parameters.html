Function Parameters (Actual and Formal)

Concept

Parameters are variables used to pass values into functions. - Actual Parameters: The values or variables given in the function call. - Formal Parameters: The variables defined in the function header that receive the values. Actual parameters are copied into formal parameters when the function is called (depending on calling method).

Actual Parameters

These are the arguments/values that are passed while calling the function. They exist in the calling function (like main()). 


// Example of Actual Parameters
#include <iostream>
using namespace std;

void show(int x) {
    cout << "Value = " << x << endl;
}

int main() {
    int n = 10;
    show(n);  // here 'n' is an actual parameter
    show(25); // here '25' is an actual parameter
    return 0;
}

Formal Parameters

These are the variables declared in the function definition that receive the values from actual parameters when the function is called. 


// Example of Formal Parameters
#include <iostream>
using namespace std;

// 'a' and 'b' are formal parameters
int add(int a, int b) {
    return a + b;
}

int main() {
    int x = 5, y = 7;
    int result = add(x, y); // x and y are actual parameters
    cout << "Sum = " << result << endl;
    return 0;
}

Key Points

- Actual parameters exist at the place where the function is called. - Formal parameters exist inside the function definition. - Number, type, and order of actual and formal parameters must match. - Actual parameters transfer their values to formal parameters when the function is invoked.

Illustration

Example showing relation between actual and formal parameters. 


#include <iostream>
using namespace std;

// formal parameters a, b
int multiply(int a, int b) {
    return a * b;
}

int main() {
    int p = 4, q = 6;
    // actual parameters p, q
    int ans = multiply(p, q);
    cout << "Product = " << ans << endl;
    return 0;
}

Notes

- Actual parameters are in the calling function; formal parameters are in the called function. - Changing a formal parameter does not normally change the actual parameter (unless using references/pointers). - This difference becomes important when studying Call by Value and Call by Reference.
