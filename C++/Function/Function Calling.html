Function Calling

Concept of Function Calling

A function call is the process of invoking a function so that its body gets executed. When a function is called, the program control transfers to the function’s definition, executes the code, and then returns back to the calling point.

Steps in a Function Call

1. Control moves from the caller to the function.
2. Arguments in the call are passed to the function’s parameters.
3. The function body executes.
4. If there is a return value, it is sent back to the caller.
5. Control resumes after the function call.

Simple Function Call

Example of calling a function with no parameters. 


// Function Calling Example
#include <iostream>
using namespace std;

void greet() { // function definition
    cout << "Hello, this is a function call!" << endl;
}

int main() {
    greet(); // function call
    return 0;
}

Function Call with Parameters

Parameters allow passing values into a function during the call. 


// Function with parameters
#include <iostream>
using namespace std;

void displaySum(int a, int b) {
    cout << "Sum = " << (a + b) << endl;
}

int main() {
    displaySum(10, 20); // function call with arguments
    displaySum(5, 7);   // another call
    return 0;
}

Function Call with Return Value

A function can return a value back to the caller. 


// Function returning value
#include <iostream>
using namespace std;

int square(int n) {
    return n * n;
}

int main() {
    int result = square(6); // function call
    cout << "Square = " << result << endl;
    return 0;
}

Nested Function Calls

A function call can be used inside another expression or another function call. 


// Nested function call
#include <iostream>
using namespace std;

int doubleValue(int x) {
    return 2 * x;
}

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(doubleValue(3), doubleValue(5)); 
    // add(6, 10) = 16
    cout << "Result = " << result << endl;
    return 0;
}

Notes

- Function calling transfers control temporarily to the function body.
- Arguments in the function call must match parameters in type, number, and order.
- Return values can be stored in variables or used directly in expressions.
- Functions can be called multiple times to reuse logic.
