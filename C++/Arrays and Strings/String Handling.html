C++

String Handling

Definition

Strings are sequences of characters used to represent text in C++.
They can be handled in two ways:
1. Using character arrays (C-style strings).
2. Using the C++ string class from the Standard Template Library (STL).

String Handling using Character Array (C-Style)

A character array is a collection of characters terminated by a null character '\0'.
These are often used in older C-style programs. 


#include <iostream>
#include <cstring>  // For string handling functions
using namespace std;

int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";

    cout << "String 1: " << str1 << endl;
    cout << "String 2: " << str2 << endl;

    // String concatenation
    strcat(str1, str2);
    cout << "Concatenated: " << str1 << endl;

    // String length
    cout << "Length of str1: " << strlen(str1) << endl;

    return 0;
}

Common String Functions (Character Arrays)

1. strlen(str) → Returns length of string.
2. strcpy(dest, src) → Copies string.
3. strcat(str1, str2) → Concatenates two strings.
4. strcmp(str1, str2) → Compares two strings. 


#include <iostream>
#include <cstring>
using namespace std;

int main() {
    char a[20] = "C++";
    char b[20];

    strcpy(b, a);  // Copy string
    cout << "Copied String: " << b << endl;

    cout << "Length of a: " << strlen(a) << endl;

    if (strcmp(a, b) == 0)
        cout << "Strings are equal" << endl;

    return 0;
}

String Handling using String Class

C++ provides the string class (in <string> header), which makes string handling easier and safer than character arrays. 


#include <iostream>
#include <string>
using namespace std;

int main() {
    string str1 = "Hello";
    string str2 = "World";

    cout << "String 1: " << str1 << endl;
    cout << "String 2: " << str2 << endl;

    // Concatenation
    string result = str1 + " " + str2;
    cout << "Concatenated: " << result << endl;

    // Length
    cout << "Length of str1: " << str1.length() << endl;

    return 0;
}

Common String Class Functions

1. length() → Returns length of string.
2. append(str) → Appends another string.
3. insert(pos, str) → Inserts string at given position.
4. erase(pos, len) → Erases part of string.
5. find(str) → Finds substring.
6. substr(pos, len) → Extracts substring. 


#include <iostream>
#include <string>
using namespace std;

int main() {
    string text = "Programming in C++";

    cout << "Length: " << text.length() << endl;

    text.append(" is fun!");
    cout << "Appended: " << text << endl;

    cout << "Substring: " << text.substr(0, 11) << endl;

    size_t pos = text.find("C++");
    if (pos != string::npos)
        cout << "Found C++ at position " << pos << endl;

    return 0;
}

Key Points

1. Character arrays (C-style strings) are older and less safe.
2. String class in C++ is easier, safer, and provides powerful functions.
3. Both methods allow string operations like copying, concatenation, comparison, and searching.
4. Prefer using string class for modern C++ programs.
