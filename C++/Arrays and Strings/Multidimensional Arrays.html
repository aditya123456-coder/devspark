Multidimensional Arrays

Definition

A multidimensional array is an array of arrays. It can have two or more dimensions.
- A one-dimensional array is a list.
- A two-dimensional array is like a table (rows and columns).
- Arrays with more than two dimensions are called multidimensional arrays.

They are useful in representing mathematical concepts such as matrices, 3D grids, or higher-dimensional data.

Declaration of Multidimensional Array

Arrays are declared by specifying the number of dimensions inside square brackets.

Syntax:
data_type array_name[size1][size2][size3]...[sizeN]; 


#include <iostream>
using namespace std;

int main() {
    int array3D[2][3][4];  // 3-dimensional array
    return 0;
}

Initialization of Multidimensional Arrays

Multidimensional arrays can be initialized during declaration or later by assigning values to each index. 


#include <iostream>
using namespace std;

int main() {
    // 2D array initialization
    int matrix[2][2] = {
        {1, 2},
        {3, 4}
    };

    // 3D array initialization
    int cube[2][2][2] = {
        {
            {1, 2},
            {3, 4}
        },
        {
            {5, 6},
            {7, 8}
        }
    };

    return 0;
}

Accessing Elements of Multidimensional Array

Elements are accessed using multiple indices. Each index corresponds to a specific dimension. 


#include <iostream>
using namespace std;

int main() {
    int cube[2][2][2] = {
        {
            {1, 2},
            {3, 4}
        },
        {
            {5, 6},
            {7, 8}
        }
    };

    cout << "Element at [1][0][1]: " << cube[1][0][1] << endl;

    return 0;
}

Traversing a Multidimensional Array

Nested loops are used to traverse multidimensional arrays. The number of loops must match the number of dimensions. 


#include <iostream>
using namespace std;

int main() {
    int cube[2][2][2] = {
        {
            {1, 2},
            {3, 4}
        },
        {
            {5, 6},
            {7, 8}
        }
    };

    cout << "Elements of 3D array: " << endl;
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            for (int k = 0; k < 2; k++) {
                cout << cube[i][j][k] << " ";
            }
            cout << endl;
        }
        cout << endl;
    }

    return 0;
}

Example: Storing Marks of Students

A 2D array can store marks of multiple students in multiple subjects. 


#include <iostream>
using namespace std;

int main() {
    int marks[3][3] = {
        {85, 90, 78}, // Student 1
        {88, 76, 92}, // Student 2
        {80, 85, 89}  // Student 3
    };

    cout << "Marks of students: " << endl;
    for (int i = 0; i < 3; i++) {
        cout << "Student " << i + 1 << ": ";
        for (int j = 0; j < 3; j++) {
            cout << marks[i][j] << " ";
        }
        cout << endl;
    }

    return 0;
}

Key Points

1. Multidimensional arrays can have two or more dimensions.
2. The most common are 2D (tables) and 3D (cubes).
3. Each additional dimension requires another index to access elements.
4. Nested loops are used for traversal.
5. They are useful in applications like matrices, 3D models, and scientific computations.
