Two-Dimensional Arrays

Definition

A two-dimensional array is an array of arrays. It is used to store data in rows and columns, similar to a matrix or table. Each element is accessed using two indices: row index and column index.

Declaration of Two-Dimensional Array

Arrays are declared by specifying the data type, array name, and dimensions (rows Ã— columns).

Syntax:
data_type array_name[rows][columns]; 


#include <iostream>
using namespace std;

int main() {
    int matrix[3][3];  // Declaration of 3x3 integer array
    return 0;
}

Initialization of Two-Dimensional Array

Arrays can be initialized during declaration or later by assigning values to each index. 


#include <iostream>
using namespace std;

int main() {
    // Initialization at declaration
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    // Another way
    int table[2][2];
    table[0][0] = 10;
    table[0][1] = 20;
    table[1][0] = 30;
    table[1][1] = 40;

    return 0;
}

Accessing Elements of Two-Dimensional Array

Elements are accessed using two indices: one for row and one for column. Indexing starts from 0 for both rows and columns. 


#include <iostream>
using namespace std;

int main() {
    int matrix[2][2] = {{10, 20}, {30, 40}};

    cout << "Element at (0,0): " << matrix[0][0] << endl;
    cout << "Element at (1,1): " << matrix[1][1] << endl;

    return 0;
}

Traversing a Two-Dimensional Array

Nested loops are used to traverse a two-dimensional array, where the outer loop iterates through rows and the inner loop iterates through columns. 


#include <iostream>
using namespace std;

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    cout << "Matrix elements: " << endl;
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }

    return 0;
}

Example: Sum of Matrix Elements

This program calculates the sum of all elements of a two-dimensional array. 


#include <iostream>
using namespace std;

int main() {
    int matrix[2][2] = {{10, 20}, {30, 40}};
    int sum = 0;

    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            sum += matrix[i][j];
        }
    }

    cout << "Sum of matrix elements = " << sum << endl;
    return 0;
}

Example: Transpose of a Matrix

Transpose of a matrix is obtained by interchanging rows and columns. 


#include <iostream>
using namespace std;

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    cout << "Transpose of matrix: " << endl;
    for (int j = 0; j < 3; j++) {
        for (int i = 0; i < 2; i++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }

    return 0;
}

Key Points

1. A two-dimensional array is like a table with rows and columns.
2. Each element is accessed with two indices.
3. Nested loops are used to traverse two-dimensional arrays.
4. Commonly used for matrices, tables, and grid-based problems.
5. Size of rows and columns must be defined at compile time (for static arrays).
