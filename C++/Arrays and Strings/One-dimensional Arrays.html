One-Dimensional Arrays

Definition

An array is a collection of elements of the same data type stored in contiguous memory locations. A one-dimensional array stores elements in a linear form, accessed using a single index. Arrays in C++ have fixed size and can store multiple values under one variable name.

Declaration of One-Dimensional Array

Arrays are declared by specifying the data type, array name, and size inside square brackets.

Syntax:
data_type array_name[size]; 


#include <iostream>
using namespace std;

int main() {
    int numbers[5];  // Declaration of integer array with size 5
    return 0;
}

Initialization of One-Dimensional Array

Arrays can be initialized during declaration or later by assigning values to each index. 


#include <iostream>
using namespace std;

int main() {
    int marks[5] = {90, 85, 78, 92, 88}; // Initialization at declaration

    // Another way
    int nums[5];
    nums[0] = 10;
    nums[1] = 20;
    nums[2] = 30;
    nums[3] = 40;
    nums[4] = 50;

    return 0;
}

Accessing Elements of Array

Array elements are accessed using their index, starting from 0 up to (size - 1). 


#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};

    cout << "First element: " << arr[0] << endl;
    cout << "Third element: " << arr[2] << endl;

    return 0;
}

Traversing a One-Dimensional Array

Arrays can be traversed using loops like for, while, or do-while. 


#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    cout << "Array elements: " << endl;
    for (int i = 0; i < 5; i++) {
        cout << arr[i] << " ";
    }
    return 0;
}

Example: Sum of Array Elements

This program calculates the sum of all elements of an array. 


#include <iostream>
using namespace std;

int main() {
    int arr[5] = {5, 10, 15, 20, 25};
    int sum = 0;

    for (int i = 0; i < 5; i++) {
        sum += arr[i];
    }

    cout << "Sum of array elements = " << sum << endl;
    return 0;
}

Example: Find Maximum Element

This program finds the largest element in the array. 


#include <iostream>
using namespace std;

int main() {
    int arr[5] = {12, 45, 23, 67, 34};
    int max = arr[0];

    for (int i = 1; i < 5; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }

    cout << "Maximum element = " << max << endl;
    return 0;
}

Key Points

1. Arrays store multiple values of the same type in a single variable.
2. Indexing starts from 0.
3. Size of the array must be defined at compile time (for static arrays).
4. Arrays can be traversed easily using loops.
5. Array size cannot be changed once declared.
