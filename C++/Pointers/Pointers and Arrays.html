Pointers and Arrays

1. Array Name as a Pointer

The name of an array acts like a pointer to its first element. So, arr is equivalent to &arr[0]. Accessing elements can be done with both array indexing and pointer arithmetic. 


#include <iostream>
using namespace std;

int main() {
    int arr[3] = {10, 20, 30};

    cout << "arr = " << arr << endl;
    cout << "&arr[0] = " << &arr[0] << endl;

    cout << "First element using arr[0]: " << arr[0] << endl;
    cout << "First element using *arr: " << *arr << endl;

    return 0;
}

2. Accessing Array Elements with Pointers

Using pointer arithmetic, we can access array elements. *(arr + i) is equivalent to arr[i]. 


#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};

    for(int i = 0; i < 5; i++) {
        cout << "arr[" << i << "] = " << *(arr + i) << endl;
    }

    return 0;
}

3. Pointer Iteration Through Array

Instead of array indexing, a pointer can be incremented to move through the array elements. 


#include <iostream>
using namespace std;

int main() {
    int arr[4] = {100, 200, 300, 400};
    int *ptr = arr;

    for(int i = 0; i < 4; i++) {
        cout << "Element " << i << " = " << *ptr << endl;
        ptr++; // move pointer to next element
    }

    return 0;
}

4. Arrays and Function Arguments

When an array is passed to a function, it decays into a pointer to its first element. So functions actually receive a pointer, not the whole array. 


#include <iostream>
using namespace std;

void printArray(int *p, int size) {
    for(int i = 0; i < size; i++) {
        cout << p[i] << " ";
    }
    cout << endl;
}

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    printArray(arr, 5);  // array passed as pointer
    return 0;
}

5. Pointer to an Array

A pointer can also point to the entire array instead of just its first element. This uses the syntax int (*ptr)[n] for an array of size n. 


#include <iostream>
using namespace std;

int main() {
    int arr[3] = {7, 8, 9};
    int (*p)[3] = &arr;  // pointer to array of 3 ints

    for(int i = 0; i < 3; i++) {
        cout << (*p)[i] << " ";
    }
    return 0;
}

6. Multi-Dimensional Arrays and Pointers

In multi-dimensional arrays, the array name acts as a pointer to the first row. We can use pointers to access elements in 2D arrays. 


#include <iostream>
using namespace std;

int main() {
    int arr[2][3] = { {1,2,3}, {4,5,6} };
    int (*p)[3] = arr;  // pointer to an array of 3 integers

    for(int i = 0; i < 2; i++) {
        for(int j = 0; j < 3; j++) {
            cout << *(*(p + i) + j) << " ";
        }
        cout << endl;
    }
    return 0;
}
