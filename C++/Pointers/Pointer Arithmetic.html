Pointer Arithmetic

1. Introduction to Pointer Arithmetic

Pointer arithmetic means performing mathematical operations on pointers. Since pointers store memory addresses, arithmetic is done in units of the data type size. For example, incrementing an int* increases its value by sizeof(int). 


#include <iostream>
using namespace std;

int main() {
    int arr[3] = {10, 20, 30};
    int *p = arr;

    cout << "Address in p: " << p << endl;
    p++; // moves to next integer
    cout << "After increment, p: " << p << endl;

    return 0;
}

2. Pointer Increment (++)

Incrementing a pointer makes it point to the next element in memory. The step size depends on the data type (for int it moves 4 bytes, for double it moves 8 bytes). 


#include <iostream>
using namespace std;

int main() {
    int numbers[3] = {100, 200, 300};
    int *ptr = numbers;

    cout << "First element: " << *ptr << endl;
    ptr++;  // move to next element
    cout << "Second element: " << *ptr << endl;

    return 0;
}

3. Pointer Decrement (--)

Decrementing a pointer moves it to the previous element in memory. This is useful when traversing arrays backwards. 


#include <iostream>
using namespace std;

int main() {
    int arr[3] = {5, 10, 15};
    int *p = &arr[2]; // points to last element

    cout << "Last element: " << *p << endl;
    p--; // move to previous element
    cout << "Second element: " << *p << endl;

    return 0;
}

4. Adding and Subtracting Integers

You can add or subtract an integer to a pointer. ptr + n moves the pointer forward by n elements, ptr - n moves it backward by n elements. 


#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int *p = arr;

    cout << "First element: " << *p << endl;
    cout << "Third element: " << *(p + 2) << endl;
    cout << "Last element: " << *(p + 4) << endl;

    return 0;
}

5. Subtracting Pointers

When two pointers point to elements of the same array, subtracting them gives the number of elements between them. 


#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int *p1 = &arr[1];
    int *p2 = &arr[4];

    cout << "p2 - p1 = " << (p2 - p1); // gives 3 elements apart

    return 0;
}

6. Pointer Comparison

Pointers can be compared using relational operators like <, >, ==. This is commonly used in loops to traverse arrays using pointers. 


#include <iostream>
using namespace std;

int main() {
    int arr[4] = {11, 22, 33, 44};
    int *start = arr;
    int *end = arr + 4;

    while(start < end) {
        cout << *start << " ";
        start++;
    }
    return 0;
}
