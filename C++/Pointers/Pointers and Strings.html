Pointers and Strings

1. String as a Character Array

In C++, a string can be represented as a character array ending with a null character '\0'. A pointer to a char can be used to access and manipulate these characters. 


#include <iostream>
using namespace std;

int main() {
    char str[] = "Hello";
    char *p = str;

    cout << "String: " << p << endl;
    cout << "First character: " << *p << endl;
    cout << "Second character: " << *(p + 1) << endl;

    return 0;
}

2. Traversing String with Pointers

A pointer can be incremented to move through each character of a string until the null character '\0' is reached. 


#include <iostream>
using namespace std;

int main() {
    char str[] = "World";
    char *p = str;

    while(*p != '\0') {
        cout << *p << " ";
        p++;
    }
    return 0;
}

3. Modifying String Using Pointers

Since strings are stored as character arrays, we can modify individual characters using a pointer. Care should be taken when modifying string literals because they are stored in read-only memory. 


#include <iostream>
using namespace std;

int main() {
    char str[] = "Cat";
    char *p = str;

    *p = 'B';   // change 'C' to 'B'
    cout << "Modified string: " << str << endl;

    return 0;
}

4. Array of Strings (Array of Pointers)

We can create an array of char* pointers, where each pointer points to a string. This is useful for storing multiple strings efficiently. 


#include <iostream>
using namespace std;

int main() {
    const char *fruits[] = {"Apple", "Banana", "Mango", "Orange"};

    for(int i = 0; i < 4; i++) {
        cout << fruits[i] << endl;
    }
    return 0;
}

5. Passing Strings to Functions Using Pointers

Strings (character arrays) can be passed to functions as pointers. This allows the function to access and manipulate the string contents. 


#include <iostream>
using namespace std;

void printString(char *s) {
    while(*s != '\0') {
        cout << *s;
        s++;
    }
    cout << endl;
}

int main() {
    char name[] = "Pointer Example";
    printString(name);
    return 0;
}
