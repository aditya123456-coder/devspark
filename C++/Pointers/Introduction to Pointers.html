Introduction to Pointers

1. What is a Pointer?

A pointer is a special variable that stores the memory address of another variable. Instead of holding a direct value, it "points" to the location in memory where the value is stored. 


#include <iostream>
using namespace std;

int main() {
    int num = 10;
    int *ptr = # // pointer storing address of num

    cout << "Value of num: " << num << endl;
    cout << "Address of num: " << &num << endl;
    cout << "Pointer holds: " << ptr << endl;
    cout << "Value using pointer: " << *ptr << endl;

    return 0;
}

2. Pointer Declaration and Initialization

A pointer is declared using the * symbol. Example: int *p; declares a pointer to an integer. Initialization is usually done using the address-of operator &. 


#include <iostream>
using namespace std;

int main() {
    int x = 25;
    int *p = &x;  // pointer initialized with address of x

    cout << "x = " << x << endl;
    cout << "p points to address: " << p << endl;
    cout << "Value at p = " << *p << endl;

    return 0;
}

3. Pointer to Pointer

A pointer can also store the address of another pointer. This is called a pointer to pointer. 


#include <iostream>
using namespace std;

int main() {
    int a = 50;
    int *p = &a;      // pointer to int
    int **pp = &p;    // pointer to pointer

    cout << "Value of a: " << a << endl;
    cout << "Value using *p: " << *p << endl;
    cout << "Value using **pp: " << **pp << endl;

    return 0;
}

4. Pointers and Arrays

The name of an array acts as a pointer to its first element. We can use pointer arithmetic to move through array elements. 


#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int *ptr = arr; // same as &arr[0]

    for(int i = 0; i < 5; i++) {
        cout << "Element " << i << " = " << *(ptr + i) << endl;
    }

    return 0;
}

5. Pointers and Functions

Pointers allow functions to modify the original variables. This is known as call by reference. 


#include <iostream>
using namespace std;

void swapValues(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 5, y = 10;
    cout << "Before swap: x=" << x << ", y=" << y << endl;

    swapValues(&x, &y);

    cout << "After swap: x=" << x << ", y=" << y << endl;
    return 0;
}

6. Null Pointer

A null pointer is a pointer that does not point to any valid memory location. It is usually initialized with NULL or nullptr in modern C++. 


#include <iostream>
using namespace std;

int main() {
    int *ptr = NULL;   // or use nullptr in C++11 and later
    if(ptr == NULL)
        cout << "Pointer is NULL (does not point anywhere)";
    return 0;
}
