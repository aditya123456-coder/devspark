Dynamic Memory Allocation (new, delete)

1. Introduction

In C++, memory can be allocated at runtime using the new operator and freed using the delete operator. This is called dynamic memory allocation. It allows creating variables, arrays, and objects when the program is running instead of fixed size at compile time.

2. Allocating a Single Variable

We can allocate memory for a single variable using new. Memory is created on the heap and accessed through a pointer. Once done, memory should be released with delete. 


#include <iostream>
using namespace std;

int main() {
    int *p = new int;  // dynamically allocate memory for int
    *p = 42;
    cout << "Value stored: " << *p << endl;

    delete p; // free memory
    return 0;
}

3. Allocating an Array

Using new, we can create arrays dynamically at runtime. Memory must be freed with delete[]. 


#include <iostream>
using namespace std;

int main() {
    int size;
    cout << "Enter size of array: ";
    cin >> size;

    int *arr = new int[size];  // dynamic array

    for(int i = 0; i < size; i++) {
        arr[i] = (i+1) * 10;
    }

    cout << "Array elements: ";
    for(int i = 0; i < size; i++) {
        cout << arr[i] << " ";
    }

    delete[] arr; // free memory
    return 0;
}

4. Dynamic Memory for 2D Arrays

We can also create 2D arrays dynamically using pointers to pointers. Memory for each row is allocated separately. 


#include <iostream>
using namespace std;

int main() {
    int rows = 2, cols = 3;
    int **matrix = new int*[rows];

    for(int i = 0; i < rows; i++) {
        matrix[i] = new int[cols];
    }

    // Assign values
    for(int i = 0; i < rows; i++) {
        for(int j = 0; j < cols; j++) {
            matrix[i][j] = (i+1) * (j+1);
        }
    }

    // Print matrix
    for(int i = 0; i < rows; i++) {
        for(int j = 0; j < cols; j++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }

    // Free memory
    for(int i = 0; i < rows; i++) {
        delete[] matrix[i];
    }
    delete[] matrix;

    return 0;
}

5. Dynamic Objects

The new operator can also be used to create objects dynamically. After use, memory is freed using delete. 


#include <iostream>
using namespace std;

class Student {
public:
    string name;
    Student(string n) { name = n; }
    void display() { cout << "Student: " << name << endl; }
};

int main() {
    Student *s = new Student("Alice"); // dynamic object
    s->display();

    delete s; // free memory
    return 0;
}

6. Difference Between delete and delete[]

  • delete → Used to free memory of a single variable or object.
  • delete[] → Used to free memory of an array.
Using the wrong form can cause memory leaks or program crashes.
