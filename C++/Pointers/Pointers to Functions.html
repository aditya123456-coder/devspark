Pointers to Functions

1. Introduction to Function Pointers

A function pointer is a pointer that stores the address of a function. Using function pointers, we can call a function indirectly through the pointer. This is useful for callbacks, passing functions as arguments, and implementing dynamic behavior. 


#include <iostream>
using namespace std;

void greet() {
    cout << "Hello from function!" << endl;
}

int main() {
    void (*fp)();   // declare a function pointer
    fp = &greet;    // store address of greet
    fp();           // call function through pointer
    return 0;
}

2. Function Pointer Syntax

The syntax to declare a function pointer: returnType (*ptrName)(parameterTypes) Example: int (*fp)(int, int); is a pointer to a function returning int and taking two int arguments. 


#include <iostream>
using namespace std;

int add(int a, int b) {
    return a + b;
}

int main() {
    int (*fp)(int, int);  // function pointer
    fp = &add;
    cout << "Sum = " << fp(5, 10);
    return 0;
}

3. Passing Function Pointers to Functions

Function pointers can be passed as arguments to other functions. This allows us to decide at runtime which function will be executed. 


#include <iostream>
using namespace std;

int add(int a, int b) { return a + b; }
int multiply(int a, int b) { return a * b; }

void compute(int x, int y, int (*operation)(int, int)) {
    cout << "Result = " << operation(x, y) << endl;
}

int main() {
    compute(5, 10, add);
    compute(5, 10, multiply);
    return 0;
}

4. Array of Function Pointers

We can store multiple function pointers in an array and choose which one to call at runtime. This is often used in menus and jump tables. 


#include <iostream>
using namespace std;

void hello() { cout << "Hello!" << endl; }
void bye() { cout << "Goodbye!" << endl; }
void welcome() { cout << "Welcome!" << endl; }

int main() {
    void (*actions[3])() = {hello, bye, welcome};

    for(int i = 0; i < 3; i++) {
        actions[i](); // call each function
    }
    return 0;
}

5. Function Pointers and Callback Mechanism

Function pointers are often used as callbacks, where one function calls another through a pointer. This allows flexible and reusable code. 


#include <iostream>
using namespace std;

void notify() {
    cout << "Task completed!" << endl;
}

void process(void (*callback)()) {
    cout << "Processing..." << endl;
    callback(); // call the callback function
}

int main() {
    process(notify);
    return 0;
}
