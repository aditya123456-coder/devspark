Introduction to C++

1. History of C++

C++ was developed by Bjarne Stroustrup at Bell Laboratories in the early 1980s. Initially, it was called "C with Classes" as it extended the C language by adding object-oriented features. In 1983, it was renamed C++. The name signifies the increment operator (++) in C, meaning an improved or extended version of C. C++ was standardized by ISO in 1998 (ISO/IEC 14882) and continues to evolve with new versions like C++11, C++14, C++17, C++20, and C++23.

2. Features of C++

  • Object-Oriented: Supports classes, objects, inheritance, polymorphism, and encapsulation.
  • Low-level Manipulation: Like C, allows direct interaction with hardware and memory.
  • Rich Library: Standard Template Library (STL) provides ready-to-use data structures and algorithms.
  • Portability: Programs can be run on different systems with little or no modification.
  • Fast and Efficient: Closer to hardware, making execution faster compared to many other high-level languages.
  • Multi-paradigm: Supports both procedural and object-oriented programming.

3. Difference between C and C++

C C++
Procedural programming language. Object-oriented programming language.
No concept of classes and objects. Supports classes, objects, inheritance, polymorphism, encapsulation.
Focuses on functions and structured programming. Focuses on objects and data abstraction.
Less secure (global data can be accessed easily). More secure (data hiding and access specifiers).
Developed by Dennis Ritchie in 1972. Developed by Bjarne Stroustrup in 1983.

4. Structure of a C++ Program

A basic C++ program consists of:
  • Header files: Provide access to input/output and other functionalities.
  • main() function: The entry point of every C++ program.
  • Statements and expressions: The body of the program.

            
#include <iostream>  // Header file for input and output
using namespace std;  // To avoid writing std:: repeatedly

int main() {
    cout << "Hello, World!";  // Output statement
    return 0;                  // Exit status
}

5. Compile and Run Process

The steps to compile and run a C++ program are:
  1. Write the program: Save the code in a file with extension .cpp.
  2. Compile: Use a compiler (e.g., g++ in GCC) to convert the source code into object code.
    Command: g++ program.cpp -o program
  3. Linking: The compiler links libraries with the object code to create an executable file.
  4. Run: Execute the output file.
    Command: ./program
