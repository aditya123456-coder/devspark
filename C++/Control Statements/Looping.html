C++

Looping Statements

while Loop

Theory: The while loop repeatedly executes a block of code as long as the condition is true. The condition is tested before each iteration, so it may execute zero times if the condition is false at the start. 


// while loop - C++
#include <iostream>
using namespace std;

int main() {
    int i = 1;
    while (i <= 5) {
        cout << "Number: " << i << endl;
        i++;
    }
    return 0;
}

do-while Loop

Theory: The do-while loop executes the block at least once, and then checks the condition at the end. This guarantees at least one execution regardless of the condition. 


// do-while loop - C++
#include <iostream>
using namespace std;

int main() {
    int i = 1;
    do {
        cout << "Number: " << i << endl;
        i++;
    } while (i <= 5);
    return 0;
}

for Loop

Theory: The for loop is used when the number of iterations is known in advance. It combines initialization, condition, and increment/decrement in one line. 


// for loop - C++
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << "Number: " << i << endl;
    }

    // Example with multiple control variables
    for (int i = 1, j = 5; i <= j; i++, j--) {
        cout << "i = " << i << ", j = " << j << endl;
    }
    return 0;
}

Notes

- Use while when the number of iterations is not fixed and depends on a condition.
- Use do-while if the block must run at least once.
- Use for when the number of iterations is known in advance.
- All three loops can be interchanged logically, but readability and intent matter.
