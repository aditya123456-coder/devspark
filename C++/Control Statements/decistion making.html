C++

Decision Making Statements

if Statement

Theory: The if statement executes a block of code only when its condition evaluates to true. If the condition is false, the block is skipped. 


// if statement - C++
#include <iostream>
using namespace std;

int main() {
    int age = 20;
    if (age >= 18) {
        cout << "You are eligible to vote." << endl;
    }
    return 0;
}

if-else Statement

Theory: The if-else provides two paths: one executes if the condition is true, and the other if it is false. 


// if-else statement - C++
#include <iostream>
using namespace std;

int main() {
    int number;
    cout << "Enter a number: ";
    cin >> number;

    if (number % 2 == 0) {
        cout << number << " is Even" << endl;
    } else {
        cout << number << " is Odd" << endl;
    }
    return 0;
}

else-if Ladder

Theory: The else-if ladder allows checking multiple conditions one after another. The first condition that evaluates true executes its block, and the rest are skipped. 


// else-if ladder - C++
#include <iostream>
using namespace std;

int main() {
    int marks;
    cout << "Enter marks: ";
    cin >> marks;

    if (marks >= 90) {
        cout << "Grade A" << endl;
    } else if (marks >= 75) {
        cout << "Grade B" << endl;
    } else if (marks >= 50) {
        cout << "Grade C" << endl;
    } else {
        cout << "Fail" << endl;
    }
    return 0;
}

Nested if Statement

Theory: A nested if means an if statement inside another if. This is used when a condition depends on another condition. 


// nested if statement - C++
#include <iostream>
using namespace std;

int main() {
    int age = 25;
    char citizen = 'Y'; // Y = yes, N = no

    if (age >= 18) {
        if (citizen == 'Y') {
            cout << "You can vote in national elections." << endl;
        } else {
            cout << "You are old enough, but not a citizen." << endl;
        }
    } else {
        cout << "You are not old enough to vote." << endl;
    }
    return 0;
}

switch-case Statement

Theory: The switch-case selects one block to execute from multiple options, based on the value of an expression. Each case represents a possible value. The break statement prevents fall-through to the next case. 


// switch-case statement - C++
#include <iostream>
using namespace std;

int main() {
    int day;
    cout << "Enter day number (1-7): ";
    cin >> day;

    switch(day) {
        case 1: cout << "Monday" << endl; break;
        case 2: cout << "Tuesday" << endl; break;
        case 3: cout << "Wednesday" << endl; break;
        case 4: cout << "Thursday" << endl; break;
        case 5: cout << "Friday" << endl; break;
        case 6: cout << "Saturday" << endl; break;
        case 7: cout << "Sunday" << endl; break;
        default: cout << "Invalid day" << endl;
    }
    return 0;
}

Notes

- Always use break in switch-case unless fall-through is desired.
- Prefer if-else when dealing with ranges; use switch for fixed constant values.
- Nesting too many if conditions can reduce readability; consider logical operators or restructuring.
